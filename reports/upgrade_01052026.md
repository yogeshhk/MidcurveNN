### Critical Fixes (3)

**1. `src/image_based/unet/train.py` — `model_from_json` removed**
- Removed `from tensorflow.keras.models import model_from_json` import (removed in Keras 3)
- `train_stage1(load=True)`: replaced `open(json) → model_from_json(json)` with `unet_stage1()` + `load_weights()`
- `train_stage2(load=True)`: same pattern, replaced with `unet_stage2()` + `load_weights()`
- Architecture functions already exist in `unet.py`; no logic change, only load mechanism fixed

**2. `src/image_based/pix2pix/main_pix2pix.py` — results grid added**
- Added `save_results_grid_images` call after training
- Loads 7 test images via `data_loader.load_data(batch_size=7, is_testing=True)`
- Generates predictions via `generator.predict()`
- Saves 3×7 results grid to `pix2pix/results/results_grid.png`

**3. `src/image_based/img2img/main_img2img_pytorch.py` — results grid added**
- Added `_tensor_to_gray_list()` helper to convert PyTorch tensors (B, C, H, W) in [-1,1] to grayscale numpy list

- Saves 3×7 results grid to `img2img/results/results_grid.png`
