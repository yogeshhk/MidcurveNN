## What Remains for Future Work

- **Graphormer backbone**: The `finetuned_graph_transformer` was built for an older HF Graphormer API. Consider migrating to a newer graph foundation model (e.g., `graphgps`, `GraphFormer` from `torch_geometric`) or vendoring the Graphormer forward code.
- **benchmark_text()**: Model inference still placeholder — needs a fine-tuned checkpoint. Run `text_based/finetuning/evaluate.py` for full Phase III evaluation.
- **CI runtime**: The geometry-based CI job installs PyTorch CPU build which may be slow on GitHub-hosted runners. Consider caching the `~/.cache/pip` directory.
