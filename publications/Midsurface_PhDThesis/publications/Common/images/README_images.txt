svg files are original vector Inkscape files
ipt files are origianl Inventor files, which can be used to generate Printscreen images, or pdfs via Drawing idw

IF YOU WANT TO ENHANCE certain images, DONT overwrite on existing one. Create a new file with new name and use that.
Some old documents may be referring this old/junk image, but need to preseve that AS IS.

Procedure to create 3D pdf images:
- Create part in inventor, put it in Clear Blue Material, save as ipt here
- Create New DRAWING (Quick Access bar, drop down menu), import the ipt
- Create New Sheet, Place view with Shaded/Wireframe option.
- Remove borders, nameplate etc
- Export it as pdf
