%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
% PDF page 65 -- Design Intent
\begin{frame}[fragile]\frametitle{Design Intent}
\begin{itemize}
\item There are several approaches you can take to create a 3D solid model, but some of them are more useful than others.
\item The most effective way to create a parametric model is the one that allows the user to update the model with the least amount of effort and time.
\item Design intent is the plan of how the model should behave when modified. It is determined by the function of the part. How will the model react if changes need to be made in the future?
\end{itemize}
\end{frame}

% PDF page 68 -- Design Intent Implementation
\begin{frame}[fragile]\frametitle{Design Intent --- Implementation}
\begin{itemize}
\item Capturing design intent is the result of knowledge, experience, skills, and practice. Design Intent is what separates a good model from a bad one.
\item Design intent can be implemented in all stages of the parametric modeling process, but the way in which constraints are applied is especially important. Take advantage of geometric constraints and equations.
\item Use your knowledge about the design to determine design intent: What features are most likely to change in the future? What condition is the design required to satisfy at all times (equally spaced holes, proportions, etc)?
\end{itemize}
\end{frame}

% PDF page 92 -- Work Plane Types
\begin{frame}[fragile]\frametitle{Work Planes --- Types}
\begin{enumerate}
\item \textbf{Offset} --- parallel plane at a fixed distance from an existing face or plane
\item \textbf{Angle} --- plane rotated about an edge by a specified angle
\item \textbf{Tangent} --- plane tangent to a curved surface
\item \textbf{Through Line Endpoint Perpendicular} --- plane through a point, perpendicular to a line
\item \textbf{Mid-plane} --- plane midway between two parallel faces
\end{enumerate}
\end{frame}
